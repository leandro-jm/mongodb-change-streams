# Welcome to **MongoDB Change Streams**!

POC using node js and change streams mongo, for monitor activities in collections.

https://www.mongodb.com/docs/manual/changeStreams/

## Requirements
- Node;
- MongoDB
  
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ node index.js
```
